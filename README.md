# Projet de Session Automne 2023 - Klemn

## Scripts Disponibles

Dans ce répertoire de projet, vous pouvez exécuter ces commandes:

### `npm start`

pour démarrer l'application React.

### `npm install`

pour installer les dépendances du serveur.

### `node ./server/serveur.js`

pour démarrer l'API.
